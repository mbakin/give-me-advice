## How to run local

Clone the project

```bash
  gh repo clone mbakin/give-me-advice
```


Go to the file directory

```bash
  cd give-me-advice
```

Open

```bash
  code .
```

Installation

```bash
  npm install
```

Start
```bash
  npm start
```
